Caracteres Comodín

Los comodines en Excel son caracteres o símbolos en Excel que remplazan uno o varios caracteres con el fin de aplicar búsqueda o filtros es más potentes en combinación de otras funciones de Excel. Los comodines en Excel se pueden implementar en los argumentos de las funciones con el fin de buscar coincidencias a partir del dato ingresado.

Hay tres caracteres comodín en Excel.

Caracter

Resultado

Signo de interrogación (?)

Este comodín se utiliza para buscar cualquier carácter.

Asterisco (*)

Este comodín se utiliza para buscar cualquier número de caracteres que precedan o sigan a cualquier carácter.

Tilde (~)

Este comodín es un carácter de escape que se utiliza antes del signo de interrogación (?) O del asterisco (*). Por ejemplo si quisiéramos buscar textos que contengan un signo de interrogación o un asterisco, entonces esta indicación se confundiría con el uso que ya tienen estos dos caracteres por ello es que este tercer carácter es útil, es decir permite identificar un signo de interrogación , asterisco o tilde como parte del patrón de identificación de los textos. Así, para buscar:

  • Un asterisco *, deberás escribir ~*,
  • Un signo de interrogación ?, escribe ~?,
  • el signo de tilde ~, escribe ~~.

Observación:No todas las funciones pueden ejecutar comodines con funciones de Excel. Aquí hay una lista de funciones que aceptan comodines.

  1. SUMAR.SI.CONJUNTO
  2. CONTAR.SI.CONJUNTO

Función SI

Existen varios tipos de funciones en Excel. Por ejemplo, podemos trabajar con funciones lógicas. Veamos un ejemplo más: la función SI es una función lógica que mostrará un dato u otro en la celda dependiendo de una condición:

SI (Condición;Verdadero;Falso)

En esta función se empieza tecleando una condición consistente en comparar dos datos.

Ejemplos:

  1. SI (B5 < 0;... (si B5 es menor que cero).
  2. SI (C7 <> B8;... (si C7 es distinto de B8).

Para realizar estas condiciones se pueden emplear los siguientes operadores de comparación:

  1. = es igual a
  2. < es menor que
  3. > es mayor que
  4. < =es menor o igual que
  5. >= es mayor o igual que
  6. < > es distinto de

Después de teclear la condición en la función SI, debemos indicar lo que debe aparecer en la celda si esa condición se cumple y, por último, separándolo con otro punto y coma, indicar lo que debe aparecer si la condición no se cumple.

Ejemplo

=SI( B5 < 0 ; "El valor es negativo" ; "El valor es positivo" )

Esta función implica que si B5 es menor que cero deberá aparecer el mensaje "El valor es negativo" y, de lo contrario, aparecerá "El valor es positivo".

Aún se puede completar más el caso: ¿y si el valor de B5 es cero? Para tenerlo en cuenta podemos incluir una función SI dentro de otra:

=SI(B5<0;”El valor es negativo”;SI(B5>0;”El valor es positivo”;”El valor es cero”))

Esta función se complementa con otras que igualmente juegan con los valores lógicos verdadero y falso:

Función O

La función O(condición 1;condición 2;...;condición n) devuelve verdadero cuando al menos una de las condiciones entre los paréntesis lo es.

Función Y

La función Y(condición 1;condición 2;...;condición n) devuelve verdadero cuando todas las condiciones entre los paréntesis lo son.

Función ESBLANCO

La función ESBLANCO(celda o expresión) devuelve verdadero cuando la celda o la expresión entre los paréntesis está vacía.

Funcion ESERROR

La función ESERROR(celda o expresión) devuelve verdadero cuando la celda o la expresión entre los paréntesis genera un error.

Funcion ESNUMERO

La función ESNUMERO(celda o expresión) devuelve verdadero cuando la celda o la expresión entre los paréntesis contiene un dato numérico.

Funcion ESTEXTO

La función ESTEXTO(celda o expresión) devuelve verdadero cuando la celda o la expresión entre los paréntesis contiene un dato de texto.
