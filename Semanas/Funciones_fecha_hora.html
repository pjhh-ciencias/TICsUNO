§ Semana 6

Funciones de fecha y hora

Fechas en Excel

Excel almacena una fecha como si fuera un número entero. Al utilizar un sistema como éste, Excel puede sumar, restar y comparar fácilmente las fechas sin necesidad de hacer alguna conversión ni cálculo especial.

Por ejemplo, la fecha 1/1/2000 está representada por el número de serie 36526 y la fecha 05/10/2000 tiene asignado el valor 36804. Como puedes imaginar, al realizar una operación con fechas se utiliza su número de serie para facilitar las operaciones

El primer y último dia de Excel

Excel reconoce todas las fechas posteriores a 1/1/1900. Es por eso que el primer día del año 1900 tiene asignado el número 1. Por el contrario, el último día que Excel reconoce es el 31/12/9999 el cual tiene asignado el número 2958465.

Valor númerico de una fecha

Es realmente sencillo conocer el valor numérico de una fecha, solamente cambia el formato de la celda que contiene una fecha a un formato General y Excel desplegará su valor numérico.

Función FECHA

La función FECHA( año ; mes ; día ) devuelve el número de serie que pertenece a una fecha especificada.

Sintaxis

FECHA( año ; mes ; día )

  • año Obligatorio. Número que representa el año de la fecha.
  • mes Obligatorio. Número que representa el mes de la fecha.
  • día Obligatorio. Número que representa el día de la fecha.

Función VALFECHA

La función VALFECHA( " dd / mm / aa " ) convierte una fecha escrita entre comillas (en formato numérico, ejemplo: “5/2/70”) en número de serie.

Función VALHORA

La función VALHORA( " hh : mm " ) convierte una hora escrita entre comillas (con formato numérico, ejemplo: “15:00”) en número de serie.

Función HOY

La función HOY( ) devuelve el número de serie de la fecha actual.

Función AHORA

La función AHORA( ) devuelve el número de serie de la fecha y hora actuales.

Función AÑO

La función AÑO( numero_serie ) convierte un número de serie en un año. El número puede ser una fecha entre comillas (por ejemplo, “29/05/68” devolvería 1968).

Función MES

La función MES( numero_serie ) dado un número de serie, se devuelve el número de día del mes correspondiente. El número puede ser una fecha entre comillas

Función DIA

La función DIA( numero_serie ), dado un número de serie, devuelve el correspondiente día del mes. El número puede ser una fecha entre comillas.

Función DIASEM

La función DIASEM( num_serie ; tipo ) convierte un número de serie de una fecha en un día de la semana (su número). Devuelve el día como un número entero entre 1 (domingo) y 7 (sábado). El segundo parámetro establece el tipo de semana (1 = semana inglesa; 2 = semana española; 3 = el primer día de la semana es el cero).

Ejemplo: La función =DIASEM( " 24 / 12 / 89 " ; 2 )" devuelve el valor 7, que corresponde a Domingo según el tipo de semana española.

Función TIEMPO

La función TIEMPO( hora ; minuto ; segundo ) devuelve el número de serie que pertenece a la hora especificada

Función HORA

La función HORA( numero_serie ) devuelve la hora de un número de serie. La hora se expresa como número entero, comprendido entre 0 (12:00 a.m.) y 23 (11:00 p.m.).

Sintaxis

HORA( numero_serie )

La sintaxis de la función HORA tiene los siguientes argumentos:

  • numero_serie Obligatorio. Es el valor de hora que contiene la hora que desea obtener. Puede escribir las horas como cadenas de texto entre comillas (por ejemplo, "6:45 p.m.", como números decimales (por ejemplo, 0,78125, que representa las 6:45 p.m.), o bien como resultado de otras fórmulas o funciones, por ejemplo VALHORA("6:45 p.m.")).

Observación: Los valores de hora son parte de un valor de fecha y se representan mediante un número decimal (por ejemplo, 0,5 representa las 12:00 p.m. porque es la mitad de un día).

Función MINUTO

La función MINUTO( numero_serie ) dado un número de serie, Devuelve los minutos de un valor de hora. Los minutos se expresan como números enteros comprendidos entre 0 y 59. El número puede ser una hora entre comillas.

Sintaxis

MINUTO( numero_serie )

La sintaxis de la función MINUTO tiene los siguientes argumentos:

  • numero_serie es el valor de hora que contiene el minuto que se desea obtener. Los valores de hora se pueden especificar como cadenas de texto entre comillas (por ejemplo, "6:45 p.m."), como números decimales (por ejemplo, 0,78125, que representa 6:45 p.m.), o bien, como el resultado de otras fórmulas o funciones (por ejemplo, VALHORA("6:45 p.m.")).

Fórmula

Descripción (Resultado)

=MINUTO("4:48:00 p.m.")

Minutos del valor de hora (48)

Función SEGUNDO

La función SEGUNDO( numero )Devuelve los segundos de un valor de hora. El segundo se expresa como número entero comprendido entre 0 (cero) y 59.

Sintaxis

SEGUNDO( numero_serie )

La sintaxis de la función SEGUNDO tiene el siguiente argumento:

  • numero_serie es el valor de hora que contiene los segundos que se desea obtener. Los valores de hora se pueden especificar como cadenas de texto entre comillas (por ejemplo, "6:45 p.m."), como números decimales (por ejemplo, 0,78125, que representa 6:45 p.m.), o bien, como el resultado de otras fórmulas o funciones (por ejemplo, HORAVAL("6:45 p.m.")).

Fórmula

Descripción (Resultado)

=SEGUNDO("4:48:18 p.m.")

Segundos del valor de hora (18)

=SEGUNDO("4:48 p.m.")

Segundos del valor de hora (0)

Función DIA.LAB

Para Excel, cada semana tiene cinco días laborables. La función DIA.LAB, excluye los fines de semana y cuenta solo los días laborables, esta una función de gran ayuda para saber exactamente la fecha en la que habrás consumido una determanada cantidad días. Por ejemplo, esto es muy útil para determinar plazos de entrega, sólo tienes que definir una fecha de inicio y después el número de días laborables. Si incluyes en la fórmula los días festivos, la función puede calcular la fecha real.

Sintaxis

DIA.LAB( fecha_inicial , dias_lab, [festivos] )

  • fecha_inicial Obligatorio. La fecha en la que se inicia el conteo.
  • dias_lab Obligatorio. La fecha donde termina el conteo.
  • festivos Opcional. Un rango donde están especificados días festivos que serán excluidos del conteo.

Atencion: Si también deseas excluir los días de vacaciones, podrás usar el 3er parámetro de la función para indicar un rango de celdas que incluya las fechas.

Ejemplo: =DIA.LAB( " 01 / 01 / 2011 " , 20 ) = 40571, que representa la fecha 28/01/2011

Función DIAS.LAB

La función DIAS.LAB( fecha_inicial ; fecha_final ; [vacaciones] ) devuelve el número de días laborables entre fecha_inicial y fecha_final, excluyendo los fines de semana y cuenta solo los días laborables.

Sintaxis

=DIAS.LAB( fecha_inicial ; fecha_final ; [vacaciones] )
  • fecha_inicial Obligatorio. La fecha en la que se inicia el conteo.
  • fecha_final Obligatorio. La fecha donde termina el conteo.
  • vacaciones Opcional. Un rango donde están especificados días festivos que serán excluidos del conteo.

Observación: Es importante tener en cuenta que Excel no contabiliza la fecha de inicio como día laborable. Es decir: si estableces el 1 de marzo como día de inicio y quieres acabar el 2 de marzo, esto implicaría solo un día de trabajo y no dos.

Función DIA.LAB.INTL

La función DIA.LAB funciona correctamente siempre y cuando se parta de que la semana tiene 5 días laborables y el fin de semana corresponde al sábado y al domingo. Sin embargo, en algunos sectores esto no es así. Para todos los que necesiten más flexibilidad en la creación de sus días laborables, Excel ofrece la función ampliada DIA.LAB.INTL, con esta función se tendrá otro parámetro disponible.

En la versión internacional se puede ajustar la duración del fin de semana y también determinar qué días de la semana deben considerarse como días laborables. Para ello, Excel pone a tu disposición dos opciones distintas. Puedes utilizar bien un número de fin de semana bien una cadena determinada.

Sintaxis

=DIA.LAB.INTL( fecha_inicial ; días ; [fin_de_semana] ; [vacaciones] )

El parámetro adicional y opcional "fin_de_semana" diferencia a esta función de DIA.LAB.

Los números de fin de semana los determina Excel. Cada número corresponde a un día de la semana o a una combinación de días de la semana:

  1. Sábado, domingo
  2. Domingo, lunes
  3. Lunes, martes
  4. Martes, miércoles
  5. Miércoles, jueves
  6. Jueves, viernes
  7. Viernes, sábado
  8. Domingo
  9. Lunes
  10. Martes
  11. Miércoles
  12. Jueves
  13. Viernes
  14. Sábado

Observaciones:

  • Si el argumento se deja vacío, Excel utiliza automáticamente la combinación de sábado y domingo.
  • Si, por lo contrario, decides utilizar una secuencia, debes seguir un esquema relativamente sencillo: la cadena consta de 7 cifras, una por cada día de la semana, comenzando por el lunes. Si se coloca un 0 en una cifra, se considerará ese día como día laborable. Si se coloca un 1, será un día libre. Por lo tanto, 1000001 se correspondería a un fin de semana de domingo y lunes. La ventaja de esta función es que puedes establecer más de dos días libres que no sean los habituales. Sin embargo, no es posible colocar un 1 en todas las cifras. Es importante tener presente que la secuencia ha de poner entre comillas.

Ejemplo: Si quieres calcular un plazo de entrega y la semana laboral va de martes a sábado, existen dos posibilidades para determinarlo con la función DIA.LAB.INTL:

  • =DIA.LAB.INTL( FECHA( 2020 ; 04 ; 06 ) ; 10 ; 2 )
  • =DIA.LAB.INTL(FECHA( 2020 ; 4 ; 6 ) ; 10 ; " 1000001")

Las dos fórmulas dan el mismo resultado. Aquí también se pueden tener en cuenta días festivos mediante referencias.

Función SIFECHA

=SIFECHA(fecha_inicio,fecha_fin,periodo)

  • Fecha Inicio: Indica la fecha a partir de donde se inicia la cuenta.
  • Fecha Fin: Marca el día final de la cuenta
  • Periodo: Sirve para indicar el periodo de medición, usar "y" para años, "m" para meses y "d" para días.

Está fórmula en muy útil cuando necesitamos encontrar el periodo de tiempo que ha pasado entre dos fecha, es decir, años, fechas o días.

No se encuentra visible en el catálogo de fórmulas en Excel, así que ten fe y verás cómo funciona.

Así que gracias a la fórmula SIFECHA podremos, por ejemplo, encontrar la edad de una persona a partir de su fecha de nacimiento o la antigüedad de un empleado en el empresa.

=SIFECHA("05/12/1995","01/06/2020","y")

Para establecer el periodo de tiempo a medir, debemos indicar "y" para años, "m" para meses y "d" para días en el último parámetro.

Valor Significado Descripción

“m” Meses Número de meses entre la fecha1 y la fecha2
“y” Años Número de años entre la fecha1 y la fecha2
“yd” Días excluyendo años Número de días entre la fecha1 y la fecha2 donde ambas fechas tienen el mismo año.
“ym” Meses excluyendo años Número de meses entre la fecha1 y la fecha2 donde ambas fechas tienen el mismo año.
“md” Días excluyendo meses y años Número de días entre la fecha1 y la fecha2 donde ambas fechas tienen el mismo mes y el mismo año.
